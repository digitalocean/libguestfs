guestfs-examples - Examples of using libguestfs from C

=head1 КОРОТКИЙ ОПИС

 #include <guestfs.h>
 
 guestfs_h *g = guestfs_create ();

 guestfs_launch (g);

або:
 cc prog.c -o prog `pkg-config libguestfs --cflags --libs`
